# Conexiencia Migración

Este proyecto corresponde a una página web estática desarrollada como parte de una entrega de diseño web. Su objetivo es presentar información de forma visual y atractiva utilizando HTML, CSS y JavaScript.

## Descripción del proyecto

La web está organizada en secciones que permiten mostrar contenido de manera clara y estructurada. Incluye elementos visuales, imágenes y una interfaz sencilla para navegar por la información.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Imágenes y recursos estáticos

## Estructura del proyecto

- conexciencia-migrada/ : carpeta principal que contiene la página web.
- index.html : archivo principal de la interfaz.
- css/ : estilos del sitio.
- js/ : lógica de interacción básica.
- img/ : recursos gráficos utilizados en la página.

## Cómo abrir el proyecto

1. Descarga o clona este repositorio.
2. Abre la carpeta conexciencia-migrada.
3. Ejecuta el archivo index.html en tu navegador.
